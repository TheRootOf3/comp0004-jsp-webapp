Welcome to the List WebApp

Quick start

There are two types of elements: a List and an Item. Lists contain only other elements. Items, however, contain elements and the content. Currently there are two content types supported by this program: Text and URL. On the next page you will see the main list. It can contain only lists.

You can add and delete elements using appropriate buttons. In order to rename an element, open it and click on its label. You can also add, delete or edit content in items. To do so, use appropriate buttons.

This app supports auto-save. However, if you want to disable it and save your data manually, you can do it on the next page
